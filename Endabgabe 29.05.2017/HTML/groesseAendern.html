Bildgröße ändern



Hier können Sie ein Bild hochladen und dieses in seiner Größe verändern.
Diese Funktion bietet sich an, wenn Sie ein Bild mit sehr breiten Maßen hochladen möchten und dieses für Ihren Anwendungszweck so nicht verwendbar ist. Das Bild wird automatisch in eine Breite von 800 Pixeln umgewandelt und hochgeladen.

Dabei wird das Bild automatisch mit der Dateiendung jpg abgespeichert. Sie müssen nicht zwangsweise eine JPEG Datei hochladen. Auch ein Bild im PNG-Format wird in ein JPEG umgewandelt.

Nach Hochladen des Bildes können Sie das Vorschaubild anklicken und das Ergebnis in einem neuen Tab öffnen.

Anmerkung: Sie werden durch ein Pop-up über den Erfolg Ihres Uploads informiert. Dies kann je nach Dateigröße ein wenig andauern.


Wählen Sie bitte die gewünschten Dateien aus, die Sie hochladen wollen
